Permanent delete (optional)

By default, Delete only hides items in this browser — they stay in the database and may reappear. To permanently remove items for good (and for every device), add a GitHub token below. This is optional.

  1. Open github.com/settings/personal-access-tokens/new
  2. Give it any name, e.g. "Opportunity Radar"
  3. Under Repository access, choose Only select repositories and pick this repo
  4. Under Permissions → Repository permissions, set Contents to Read and write
  5. Click Generate token, copy it, and paste it below
