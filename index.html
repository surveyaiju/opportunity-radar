How it works

The GitHub Action runs daily at 6 AM UTC.
It collects from RSS feeds + DuckDuckGo search, runs Gemini AI classification, deduplicates, and commits data/opportunities.json.

This dashboard just reads that JSON file — no server needed.

Google Alerts → RSS

Google Alerts must deliver as RSS feeds, not email, to be ingested by the collector.

To convert: alerts.google.com → Edit alert (pencil) → Deliver to: RSS feed → Save → copy the URL → add it to config/sources.json under rss_feeds.

Extra feeds (dashboard only)

These feeds display here but are not collected by the daily job. To add to the daily job, edit config/sources.json.
